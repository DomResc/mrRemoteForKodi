# mrRemote for Kodi

**mrRemote for Kodi** is a Kodi remote control for the **UWP** devices. It’s really easy to use and with the step by step guide you can configure Kodi in less than 5 minutes. The UI, developed with UWP platform in mind,is easy to navigate and looks great on a variety of devices and screen sizes. With mrRemote for Kodi you can control how many Kodi media center you want, without any limitations.
And now finally it's on **GitHub**.

Included features:
- Control Kodi with an easy to use remote
- Complete library Management
- Simply add a mediacenter using the wizard
- Wake up your mediacenter with Wake-on-Lan
- Play and Cast from your Kodi mediacenter
